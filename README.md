# BebraShield 4 — Настройки ядра

Актуальные настройки ядра нашего сервера **БебраШилд**. 

Репозиторий создан специально для тех, кто хочет локально протестировать работоспособность ферм/редстоун-машин, проверить поведение сервера или помочь с оптимизацией.

---

## Как запустить локальный сервер у себя

Если ты хочешь потестить свои сборки в условиях, максимально приближенных к боевым:

1. Скачай этот репозиторий (проще всего: зелёная кнопка **Code** -> **Download ZIP**).
2. Распакуй всё это дело в отдельную папку.
3. Убедись, что у тебя установлена нужная Java под нашу версию.
4. Запусти файл `start.bat`. 
5. Сервер сгенерирует чистый мир с нашими настройками ядра и лимитами.

## Остальная информация
Версия `26.1.2`. Java `25+`. Ядро [Purpur](https://purpurmc.org/download/purpur), билд #2592 от 15.06.2026. Здесь лежит только "скелет" сервера: стартап, файл ядра и его конфигурации, ничего лишнего. Присоединиться к локальному серверу можно через айпи `localhost` или `127.0.0.1`.
